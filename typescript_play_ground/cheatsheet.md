# TypeScript Guide

## TypeScript Development Env
- Online compiler: [TypeScriptLang](www.typescriptlang.org/play/index.html)
- Local compiler:
    - npm install -g typescript
    - tsc --version
    - tsc main.ts


> npm install -g typescript

> tsc --version

> tsc main.ts

> node main.js

> tsc main.ts | node main.js

> tsc main.ts && node main.js
